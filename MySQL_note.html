MySQL笔记

一、数据库概述

首先要搞明白三个概念！

  1. 数据库：Database ，简称DB，就是一堆按照一定格式存储的文件

  2. 数据库管理系统：Database Management System，简称DBMS，负责管理数据库。

    常用的数据库管理系统：MySQL、SQL server

  3. SQL：结构化查询语言，用来进行增删查改

三者关系：

DBMS --> 执行 --> SQL --> 操作 --> DB

二、MySQL基本操作

启动MySQL：net start MySQL

停止MySQL：net stop MySQL

登陆MySQL（会显示密码）：mysql -uroot -p你的密码

登陆MySQL（不会显示密码）：mysql -uroot -p，回车后输入密码

退出MySQL：exit

查看数据库：show databases;

选择数据库：use 数据库名字;

创建数据库：create database 数据库名字;

三、表

表是数据库中最基本的数据单元！

所以表是什么？

姓名性别年龄
张三男18
李四男19
王五女20

以上这个表格就是一张表。

下面是关于表结构的概述

姓名性别年龄
张三男18---->行（记录）
李四男19---->行（记录）
王五女20---->行（记录）
列（字段）****列（字段）列（字段）

任何一张表都有行和列，行被称为数据或者记录，列被称为字段。

字段都有字段名、数据类型、约束等属性。

  1. 字段名：就是字段的一个名字。
  2. 数据类型：字符串、数字
  3. 约束：有很多，最常见的一个叫唯一性约束，添加后该字段的数据不能重复。

如何查看数据库有哪些表？

show tables;

四、SQL的分类

  1. DQL（数据查询语言）

    凡是带有select的都是查询语句

    select...

  2. DML（数据操作语言）

    凡是对DB进行增删改的都是操作语句

    insert增、delete删、update改

  3. DDL（数据定义语言）

    凡是带有create、drop、alter的都是定义语句

    DDL操作的是表的结构（例如删掉整行整列），不是表的数据

    create增、drop删、alter改
